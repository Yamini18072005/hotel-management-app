# Hotel Management System

This is a simple Hotel Management backend project developed using Node.js and Express.
This project was created as part of an internship to understand REST APIs.

## Features
- View available rooms
- Book a room
- View bookings
- REST API using GET and POST methods

## Technologies Used
- Node.js
- Express.js
- Postman
- GitHub

## API Endpoints

### GET Rooms
GET /rooms

### POST Book Room
POST /book

### GET Bookings
GET /bookings

## How to Run
1. Install Node.js
2. Open project folder in VS Code
3. Run the following commands:

npm install  
node index.js  

Server will start on port 3000
